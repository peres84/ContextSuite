## Constraint: Primary brand color must stay green

Source: Brand review board, 2026 Q1

The primary brand color for the Green Guard Studio website must remain green.

The approved primary green is `#2f9e44`. This rule applies to:

- hero accents
- primary CTA buttons
- badges and highlight chips
- navigation emphasis
- section dividers and callout styling

Do not switch the global theme or the primary UI color from green to red.

Red is reserved only for:

- destructive actions
- validation errors
- alert banners

Reason:

Green is part of the visual identity and tested better in previous release reviews.
