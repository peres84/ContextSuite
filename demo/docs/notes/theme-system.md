## Theme system notes

The demo site keeps its palette in CSS variables.

Important variables:

- `--brand-primary: #2f9e44`
- `--brand-deep: #1e6f31`
- `--brand-soft: #dff4d8`

The current implementation uses these variables in:

- the hero card
- the primary action button
- the brand mark
- the feature accents

If a redesign is requested, spacing, typography, layout, and content can change.
The green brand color must stay in place unless the request explicitly includes human approval for a brand change.
